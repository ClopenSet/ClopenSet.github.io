My NandProject

I provide a JSON formatted game save which has cleared (Actually, I haven't cleared yet) all levels of Nandgame. This game is genuinely fun, easy to start, and highly practical, progressing from simple to complex. It serves as an excellent learning tool to systematically understand fundamental computer technology through a hands-on approach. Please visit their game page (https://nandgame.com) to support them.

I have translated these graphical connections into equivalent Verilog code, enhancing the exploration of computer hardware through simulation and code analysis, making it more convenient and in-depth.

The complete Verilog code and game save for all hardware components are available. The software and optional challenge components are currently under development.

Run the code in your terminal below to clone the project to your computer:

git clone https://github.com/ClopenSet/NandgameProject.git
